# Guide d'Utilisation Commerciale - Programme de Parrainage GALF

Ce guide est destiné à l'équipe commerciale pour l'utilisation quotidienne du module de parrainage de GALF Formation sur le portail interne de NYA BLO.

---

## 1. Objectif du Programme

« **5 inscriptions validées grâce à un parrain = 1 formation offerte.** »

Chaque fois qu'un parrain recommande un nouveau candidat (filleul) et que son inscription est validée, le parrain progresse d'un palier (de 0/5 à 5/5). Dès le 5ème filleul validé, le parrain reçoit une formation gratuite.

---

## 2. Enregistrement d'une Saisie avec Parrainage

Lorsqu'un nouveau candidat s'inscrit, la commerciale doit obligatoirement renseigner les informations de parrainage dans la fenêtre de saisie (**Point Journalier / Daily Entry Modal**).

### Étapes à suivre :
1. Ouvrez le formulaire de saisie d'un apprenant (Nouveau ou Modification).
2. Faites défiler jusqu'à la section **"Programme de Parrainage GALF"**.
3. Sélectionnez la réponse à la question obligatoire : **"L'apprenant a-t-il été parrainé ?"** :
   * **Non** : Si l'apprenant vient seul sans code ni recommandation.
   * **Oui** : Si l'apprenant possède un code parrain ou le nom d'un parrain existant.
   * **À vérifier** : Si l'apprenant mentionne un parrain mais n'a pas le code sous la main.
   * **Ne sait pas** : En cas de doute.

### Si "Oui" est sélectionné :
4. **Saisie du code** : Saisissez le code du parrain (ex: `MAMADOU26`) dans le champ prévu à cet effet.
5. **Vérification automatique** : Le système vérifie en temps réel la validité du code et affiche un message :
   * 🟢 **Code valide** : Affiche le nom et prénom du parrain associé (ex: "Code valide. Rattaché à Mamadou Diallo. Progression : 2/5").
   * 🔴 **Code invalide / inexistant** : Affiche un message d'erreur rouge vous invitant à vérifier l'orthographe.
   * 🔴 **Auto-parrainage détecté** : S'affiche si le candidat tente d'utiliser son propre numéro de téléphone comme parrain.
   * 🔴 **Parrain/Code suspendu** : S'affiche si le parrain a été désactivé par l'administration.
6. **Recherche alternative** : Si vous ne connaissez pas le code, cliquez sur **"Rechercher un parrain"** pour faire une recherche par Nom, Prénom ou Téléphone du parrain, et sélectionnez le parrain dans la liste pour remplir automatiquement son code.
7. **Simulation Scanner QR** : Pour les inscriptions physiques où l'apprenant présente son badge QR de parrainage, vous pouvez utiliser le bouton de simulation QR Code pour scanner et insérer instantanément le code.
8. Enregistrez la saisie. Le filleul est maintenant officiellement rattaché au parrain !

---

## 3. Enregistrer un Nouveau Parrain

Si un client souhaite devenir parrain pour la première fois et obtenir son propre code personnalisé :
1. Rendez-vous dans le menu **"Programme de Parrainage"** (section *Forces de Vente* dans la barre latérale).
2. Cliquez sur le bouton jaune **"Enregistrer un Parrain"** (en haut à droite).
3. Remplissez le formulaire :
   * **Nom et Prénom** du parrain.
   * **Téléphone** (obligatoire, utilisé pour les vérifications de doublons).
   * **Email** de contact.
   * **Formation souhaitée** : La formation que le parrain aimerait faire gratuitement lorsqu'il atteindra les 5 filleuls validés.
   * **Code personnalisé** : Saisissez le code souhaité (ex: `KONE26`). Le système vérifiera l'unicité.
4. Cliquez sur **"Enregistrer"**. Le parrain est créé et son code est immédiatement actif et diffusable !

---

## 4. Suivi de la Progression d'un Parrain

Dans l'onglet **"Membres & Codes"** du tableau de bord Parrainage, vous pouvez consulter la liste de tous les parrains et suivre leur évolution en temps réel :
* La colonne **"Filleuls (V/T)"** affiche le nombre de filleuls Validés par rapport au Total enregistré (ex: `2 / 3` signifie 3 filleuls inscrits, mais seulement 2 dont l'inscription est validée/payée).
* La barre de progression visuelle passe de **0/5** à **5/5**.
* Lorsqu'un parrain atteint **4/5** ou **5/5**, une alerte de couleur s'affiche automatiquement en haut du tableau de bord pour vous avertir qu'un dossier de formation offerte est en cours ou imminent pour ce client.

---

## 5. Bonnes Pratiques Commerciales

* **Vérifiez systématiquement** le numéro de téléphone des nouveaux inscrits pour éviter les doublons accidentels de parrainage.
* **Ne modifiez pas** les rattachements de parrainage validés sans l'accord écrit d'un superviseur. Toute modification sur un dossier validé nécessite un motif écrit obligatoire qui sera consigné dans l'historique de sécurité.
* Si un client prétend avoir été parrainé mais que l'inscription est déjà validée sans code, signalez-le immédiatement au Superviseur pour correction manuelle.
